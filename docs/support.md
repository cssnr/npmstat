## GitHub Discussions

If you have a question or need help with anything,
start a discussion in [General](https://github.com/cssnr/npmstat/discussions/categories/general)
or [Q&A](https://github.com/cssnr/npmstat/discussions/categories/q-a).

If you want to see a new feature, start a discussion in
the [Feature Requests](https://github.com/cssnr/npmstat/discussions/categories/feature-requests) section.

[:fontawesome-brands-github: Start A Discussion](https://github.com/cssnr/npmstat/discussions){ .md-button .md-button--primary }

## GitHub Issues

If you run into any problems or find a bug, please [open an issue](https://github.com/cssnr/npmstat/issues).

**These reports are treated with the highest priority.**
